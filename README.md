# Re Carlo V Coach

Applicazione web installabile, local-first, per pianificare e registrare allenamenti endurance, forza e hybrid training.

## Privacy

Questo repository contiene soltanto la shell statica dell'applicazione. Non contiene profili atleta, storico degli allenamenti, esportazioni WHOOP o Strava, token OAuth o credenziali private.

I dati dell'atleta restano nel browser e, quando l'utente attiva volontariamente la sincronizzazione, nel suo account cloud protetto.

## Utilizzo

L'app è pubblicata tramite GitHub Pages. La prima apertura mostra un profilo vuoto da configurare oppure consente di accedere a un account cloud esistente.
